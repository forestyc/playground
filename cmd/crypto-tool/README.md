# CRYPTO-TOOL

A sm4 crypto tool.
```shell
$ crypto-tool.exe --help
Usage of D:\code\playground\cmd\crypto-tool\crypto-tool.exe:
  -encoding string
        base64 encoding, std/url, use url by default
  -key string
        key
  -method string
        encrypt/decrypt
  -text string
        Cipher text/Plain text
```
